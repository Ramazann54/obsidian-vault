## БАЗА НЕЙРОСЕТЕЙ

- **Нейросеть** → аппроксимация сложных функций
    
- **Веса** → то, что обучается
    
- **Функция активации** → **нелинейность**
    
- **Без активации** → сеть = линейная модель
    

---

## 🔹 ОБУЧЕНИЕ

- **Loss function** → измеряет ошибку
    
- **Backpropagation** → распространение ошибки назад
    
- **Gradient descent** → минимизация loss
    
- **Исчезающий градиент** → градиенты → 0 → ранние слои не учатся
    
- **ReLU** → не насыщается → градиент жив
    

---

## 🔹 DATA SPLIT

- **Train set** → обучение
    
- **Test set** → **оценка обобщающей способности**
    
- **Overfitting** → хорошо на train, плохо на test
    

---

## 🔹 CNN (СВЁРТОЧНЫЕ СЕТИ)

- **Изображение** → локальная структура
    
- **CNN** → учитывает локальность
    
- **Свёртка** → одно ядро, много позиций
    
- **Shared weights** → меньше параметров
    
- **Pooling** → уменьшение пространственного размера
    
- **Padding** → сохранить размер карты признаков
    

---

## 🔹 RNN / LSTM

- **Последовательность** → порядок важен
    
- **RNN** → проблемы с длинными зависимостями
    
- **Vanishing / exploding gradient** → во времени
    
- **LSTM / GRU** → **память** → длинные зависимости
    

---

## 🔹 SEQ2SEQ

- **Encoder** → кодирует вход
    
- **Decoder** → генерирует выход
    

---

## 🔹 ATTENTION

- **Attention** → фокус на важном
    
- **Self-attention** → токены смотрят друг на друга
    
- **Почему O(n²)** → каждый с каждым
    

---

## 🔹 TRANSFORMER

- **Transformer** → attention + FFN
    
- **Нет RNN** → параллельность
    
- **Positional encoding** → порядок токенов
    
- **Feed-forward** → применяется **к каждому токену отдельно**
    

---

## 🔹 LLM

- **Токен** → минимальная единица текста (подслово)
    
- **Autoregressive** → токен за токеном
    

### Модели:

- **BERT** → encoder-only → **понимание**
    
- **GPT** → decoder-only → **генерация**
    
- **Masked self-attention** → нельзя видеть будущее
    

---

## 🔹 ОБУЧЕНИЕ БОЛЬШИХ МОДЕЛЕЙ

- **Pretraining** → обучение на больших данных
    
- **Fine-tuning** → дообучение под задачу
    

---

## 🔹 ЭТИКА ИИ

- **Bias** → смещения
    
- **Цель этики ИИ** → минимизация вреда
